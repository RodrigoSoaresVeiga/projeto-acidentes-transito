Análise de Acidentes de Trânsito no Brasil

Projeto analítico desenvolvido com dados simulados entre 2015 e 2024, explorando padrões de acidentes de trânsito no Brasil. O estudo aborda regiões críticas, evolução temporal, horários de risco, condições climáticas e tipos de ocorrência para apoiar análises e tomadas de decisão.

Acessar Dashboard Abrir Notebook
2015–2024 Período analisado
5 Regiões brasileiras
37 Cidades monitoradas
6 KPIs principais

Atividades Desenvolvidas

  • Dashboard interativo utilizando Streamlit
  • Análise exploratória de dados em Notebook
  • Tratamento e organização da base CSV
  • Criação de visualizações analíticas
  • README com instruções de execução e publicação

Resumo Conclusivo

O projeto demonstra como técnicas de análise e visualização de dados podem auxiliar na compreensão de padrões relacionados à segurança viária. Os resultados permitem identificar regiões mais críticas, horários com maior incidência de acidentes e fatores ambientais que influenciam diretamente os índices analisados.
